# DevToys

A flutter implement of [DevToys[microsoft]](https://github.com/veler/DevToys) .


## Supported Tools

  * Converters
    * JSON<>YAML converter
    * Number Base Converter
  * Encoders/Decoders
    * Base64 Encoder/Decoder
    * URL Encoder/Decoder
    * GZip Encoder/Decoder
    * JWT Decoder
  * Formatters
    * JSON formatter
    * Dart formatter
  * Generators
    * Hash Generator
    * UUID Generator
    * Lorem Ipsum Generator
  * Text
    * Markdown Preview
    * Regex Tester
