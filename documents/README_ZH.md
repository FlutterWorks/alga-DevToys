# Alga

一款强大的开发者工具集.

> inspired by [DevToys](https://github.com/veler/DevToys).

## 支持的工具

  * Converters
    * JSON<>YAML converter
    * Number Base Converter
  * Encoders/Decoders
    * Base64 Encoder/Decoder
    * URL Encoder/Decoder
    * GZip Encoder/Decoder
    * JWT Decoder
    * Uri Parser
  * Formatters
    * JSON formatter
    * Dart formatter
  * Generators
    * Hash Generator
    * UUID Generator
    * Lorem Ipsum Generator
    * Sass to Css Generator
  * Text
    * Markdown Preview
    * Regex Tester
  * Image Tools
    * Blurhash Tool
    * QR Code Tool
  * Server Tools
    * Static Server Tool

## 开始构建

### 构建前

#### Android

* place your own key and modify this file (`android/key.properties.example`) and rename to `key.properties`

### 构建App

```shell
flutter_distributor release --name release
```

> will generate apps at `dist/`
